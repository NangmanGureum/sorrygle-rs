# sorrygle-rs

This is a [Sorrygle] (https://github.com/JJoriping/Sorrygle.git) compiler written in Rust

## How to use

You can run command to simple text to play:
```bash
sorrygle-rs play "(q=8)cdefgfedc~~~"
```

or convert text file to midi file
```bash
sorrygle-rs -c script.txt output.mid
```
